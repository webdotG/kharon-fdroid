После добавления репозитория F-Droid будет автоматически показывать обновления.

## О приложении

**Kharon Messenger** - защищённый мессенджер.

- Минимальная версия Android: 10 (API 29)
- Исходный код: [github.com/webdotG/kharon-mess](https://github.com/webdotG/kharon-mess)

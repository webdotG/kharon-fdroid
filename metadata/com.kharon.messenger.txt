Description:|
    Kharon — минималистичный E2E зашифрованный мессенджер для небольших команд.

    Сообщения исчезают через 2 минуты после прочтения.
    Никаких аккаунтов, телефонных номеров, истории переписки.

    Особенности v2.0:
    * Reception Mode — выбирай режим связи: всегда онлайн или окно раз в N минут
    * PULSE режим — AlarmManager будит приложение точно по расписанию
    * Динамический TTL — сервер хранит сообщения столько сколько нужно для твоего окна
    * 4 темы — Princess, Terminal Dark, Terminal Light, Shadow
    * Счётчик непрочитанных сообщений
    * Статусы сообщений — отправлено, доставлено, прочитано
    * Credits система — защита от спама
    * Запуск после перезагрузки

    Шифрование: X25519 + XSalsa20-Poly1305 (libsodium)
    Сервер: self-hosted Node.js relay, никогда не видит plaintext
